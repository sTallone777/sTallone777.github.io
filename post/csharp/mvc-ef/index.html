How to create entity in dotnet core

Install Packages

First of all, you need install the packages that needed to create the entitys through the nuget package manager.

  • Microsoft.EntityFrameworkCore
  • Microsoft.EntityFrameworkCore.Tools
  • Microsoft.EntityFrameworkCore.Design
  • Microsoft.EntityFrameworkCore.Relational
  • Pomelo.EntityFrameworkCore.MySql

Create Entity

Then you can create entitys with the following command at the console.

For Mysql:

Scaffold-DbContext -Provider Pomelo.EntityFrameworkCore.MySql -Context DBContext -Project Common -Connection "Server=192.168.10.100;User Id=ranbo;Password=ranBo000;Database=FILECTLSYS" -outputdir DBModel

For Sql:

Scaffold-DbContext -Force “Server=192.168.10.100;Database=FILECTLSYS;User ID=ranbo;Password=ranBo000;” Microsoft.EntityFrameworkCore.SqlServer -OutputDir DBModel
